**1. Create TabSidebarView.xaml:**
Vertical sidebar shell. Background #191919. Fixed width 260px. Contains:
- A [+] New Tab button at the top (same style as TabBarView's AddTabButton: "+" content, #999999 foreground, transparent background, 32px height, full width)
- A ScrollViewer with VerticalScrollBarVisibility="Auto" wrapping a StackPanel (x:Name="TabList", Orientation="Vertical") for programmatic tab entry insertion

**2. Create TabSidebarView.xaml.cs:**
Follow the exact pattern of TabBarView.xaml.cs but with vertical layout and richer entries.

Public API (must match what MainWindow expects):
- `public event Func<Task>? NewTabRequested;`
- `public void Attach(TabViewModel viewModel, AttentionStore? attentionStore, Dictionary<string, WorkspaceView> tabViews)` -- note the additional tabViews parameter to enable preview text access

Internal state:
- `_viewModel`, `_attentionStore`, `_tabViews` references
- `_attentionForeground` = SolidColorBrush #3282F0 (blue)
- `_defaultForeground` = SolidColorBrush #CCCCCC
- `_tabBlinkTimers` dictionary (same pattern as TabBarView)
- `_refreshTimer`: DispatcherTimer at 2-second interval for preview text refresh

Event subscriptions (in Attach):
- `TabStore.TabsChanged` -> re-render
- `TabStore.ActiveTabChanged` -> re-render
- `AttentionStore.AttentionChanged` -> re-render

RenderTabs() method -- builds tab entries programmatically:
For each tabId in TabStore.TabOrder:
  - Get tab label from TabStore.GetTab(tabId).Label
  - Get workspace from _viewModel.GetWorkspace(tabId)
  - Get active pane ID from workspace.LayoutStore.ActivePaneId
  - Get cwd: workspace.GetSessionForPane(activePaneId)?.LastKnownCwd ?? ""
  - Get preview text: access WorkspaceView's TerminalPaneView.Bridge.GetRecentLines(2) for the active pane. To access this, WorkspaceView needs to expose a method. Add `public string[] GetPreviewText(string paneId, int lineCount)` to WorkspaceView that checks _paneViews for the paneId, gets the Bridge, and calls GetRecentLines. If pane not found or bridge is null, return empty array.
  - Determine isActive and hasAttention (same logic as TabBarView)
  - Call CreateSidebarTabEntry(tabId, label, cwd, previewText, isActive, hasAttention)

CreateSidebarTabEntry method -- builds a Grid per tab:
- Outer Grid: full width, padding (12, 8, 8, 8)
- Background: active=#2D2D2D, inactive=transparent
- Two columns: star (text content) + auto (close button)
- StackPanel in column 0 with three rows:
  1. Title row: horizontal StackPanel with optional blue dot (Unicode bullet \u25CF + space) + title TextBlock (12px, white #CCCCCC or blue #3282F0 if attention, CharacterEllipsis trimming)
  2. CWD line: TextBlock (10px, #808080, CharacterEllipsis)
  3. Preview: TextBlock (10px, #606060, MaxLines=2, NoWrap, CharacterEllipsis). Join preview lines with \n.
- Close button (column 1): "X" text, 20x20, transparent bg, #888888 fg, Visibility=Collapsed by default
- PointerEntered on outer Grid -> close button Visibility=Visible
- PointerExited on outer Grid -> close button Visibility=Collapsed
- PointerPressed on outer Grid -> _viewModel.SwitchTab(tabId)
- closeButton.Click -> _viewModel.CloseTabAsync(tabId)
- closeButton.PointerPressed -> e.Handled = true (prevent bubble)

Right-click context menu (ContextFlyout on the outer Grid):
- MenuFlyout with "Rename" MenuFlyoutItem
- Rename click -> show inline TextBox rename (same pattern as TabBarView.StartInlineRename but adapted for sidebar layout: replace the text StackPanel content with a TextBox, commit on Enter/LostFocus, cancel on Escape, re-render on commit/cancel)

Attention blink animation (port from TabBarView):
- StartTabBlinkAnimation: DispatcherTimer at 500ms, 8 toggles (4 blinks), then steady blue on both dot and title TextBlock
- Store timer in _tabBlinkTimers keyed by tabId
- On re-render, stop all existing timers first (same as TabBarView)

Preview refresh timer:
- Start 2-second DispatcherTimer in Attach
- On tick: call RenderTabs() to refresh all entries (picks up new preview text, process names, cwd changes)
- Stop timer in a Detach() method

Add Detach() method:
- Stop refresh timer
- Unsubscribe from TabStore and AttentionStore events
- Stop all blink timers

**3. Add GetPreviewText to WorkspaceView.xaml.cs:**
Add public method:
```csharp
public string[] GetPreviewText(string paneId, int lineCount)
{
    if (_paneViews.TryGetValue(paneId, out var paneView) && paneView.Bridge is not null)
    {
        return paneView.Bridge.GetRecentLines(lineCount);
    }
    return Array.Empty<string>();
}
```
