  <files>src/Wcmux.Core/Terminal/AnsiStripper.cs, tests/Wcmux.Tests/Terminal/AnsiStripperTests.cs</files>
  <behavior>
    - Strip("hello") -> "hello" (plain text unchanged)
    - Strip("\x1b[31mred\x1b[0m") -> "red" (CSI color codes removed)
    - Strip("\x1b]0;window title\x07rest") -> "rest" (OSC sequences removed)
    - Strip("\x1b]0;title\x1b\\rest") -> "rest" (OSC with ST terminator)
    - Strip("\x1b(B\x1b[m") -> "" (charset designation + CSI)
    - Strip("hello\x01\x02world") -> "helloworld" (control chars stripped, but \n \r \t kept)
    - Strip("\x1b[1;32mPS C:\\Users>\x1b[0m ") -> "PS C:\\Users> " (real PowerShell prompt)
    - Strip("") -> "" (empty input)
    - Strip(text with mixed content) preserves all printable characters and whitespace
  </behavior>
  <implementation>
    Create src/Wcmux.Core/Terminal/AnsiStripper.cs as a static class with a single compiled Regex
    that matches: CSI sequences (ESC[ params letter), OSC sequences (ESC] ... BEL or ST),
    charset designations (ESC( or ESC) followed by char), two-char ESC sequences, bare ESC,
    and control chars except \n \r \t. Use RegexOptions.Compiled for performance.
  </implementation>
</feature>

<feature>
  <name>Ring buffer in TerminalSurfaceBridge</name>
  <files>src/Wcmux.Core/Terminal/TerminalSurfaceBridge.cs, tests/Wcmux.Tests/Terminal/OutputRingBufferTests.cs</files>
  <behavior>
    - After output batch is drained, stripped text is split into lines and stored in ring buffer
    - Ring buffer capacity is 20 lines (configurable via constructor parameter)
    - GetRecentLines(2) returns the last 2 non-empty lines
    - GetRecentLines(5) when only 3 lines buffered returns 3 lines
    - Ring buffer wraps around when capacity exceeded (oldest lines overwritten)
    - Empty/whitespace-only lines are not stored in the buffer
    - Thread-safe: concurrent writes (batch loop) and reads (UI timer) do not corrupt data
    - GetRecentLines returns empty array when no output has been captured
  </behavior>
  <implementation>
    Add to TerminalSurfaceBridge: private string[] _ringBuffer, int _ringHead, int _ringCapacity (default 20),
    object _ringLock. Add private AppendToRingBuffer(string rawBatch) that calls AnsiStripper.Strip,
    splits on \n, trims, skips empty, writes to circular buffer. Add public string[] GetRecentLines(int count).
    Call AppendToRingBuffer(batch) in RunOutputBatchLoop after bell detection, before _writeToSurface.
    Add optional ringBufferCapacity parameter to constructor (default 20).
  </implementation>
</feature>
